This project provides a complete analysis of books in the market, from data cleaning to actionable insights. The dashboard allows comparison of books based on multiple factors such as prices, ratings, and awards.

Key Dimensions:

Number of books and awards by author.

Number of books published over time (years).

Book prices by title.

Marketing spend by publisher.

Table showing books by genre and their ratings.

Features:

Interactive dropdown filters for City, Country, and Published Year.

Data modeling to structure and connect datasets.

Analysis to provide clear insights for decision-making.

Tools Used: 

Microsoft Excel / Power BI / Data Modelling / Power Pivot .
